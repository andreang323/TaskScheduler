# TaskScheduler

## Dependencies

- This project was configured to use the Jetbrains IntelliJ build system.
- Z3 Prover must be installed and on your PATH. Microsoft Z3 is available from [https://github.com/Z3Prover/](https://github.com/Z3Prover/z3/releases).
  > It is possible to configure the path only for a specific application in the IntelliJ launch configurations menu
- Java must be configured to allow non-Java dependencies with the following or comparable VM option: `--enable-native-access=ALL-UNNAMED`.
- Some UI in this project was configured using the Jetbrains Swing UI Designer extension, which works best with the IntelliJ build system.
