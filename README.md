# TaskScheduler

## Dependencies

- This project was configured to use the Jetbrains InteliJ build system.
- Z3 Prover must be installed and on your path. Microsoft Z3 is avaiable from [https://github.com/Z3Prover/](https://github.com/Z3Prover/z3/releases).
  > It is possible to configure the path only for a speicf application in the InteliJ launch configurations menu
- Java must be configured to allow non-Java dependancies with the following or comparaable VM option: `--enable-native-access=ALL-UNNAMED`.
- Some UI in this project were configured useing the Jetbrains Swing UI designer extension, which works best with the InteliJ build system.
